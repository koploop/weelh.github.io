使用命令行快速生成项目目录结构. 可以定义目录深度.

使用tree命令生成树状目录

之前一直看到别人博客里面, 显示文档目录的时候很整洁,但是自己一直不知道怎么生成的,现在终于知道是用tree这个神奇;

  1. 前提是Mac已经安装了homebrew这个插件管理器,没有的话,还是建议装上,炒鸡方便;没有安装? 一句命令搞定.
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)
  2. 安装tree
    brew install tree
  3. CD到想要输出文档目录的地方,执行命令tree, 会把当前目录所有深度下的文件都遍历出来,但是我们通常只用输出目录,下面就用到了几个常用的命令:
    tree -d => 这里-d指的是directory，也就是只输出目录路径格式;
    tree -L 2 => 输出文件夹遍历层数;
