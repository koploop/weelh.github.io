增删改查Linux命令

2017-04-17

mkdir

可选参数

  • -p (— parents) 若路径中目录不存在,则创建该目录.

可以递归建立那些尚不存在的目录,即一次建立多个

Ex: mkdir -p submodule/test 一次创建submodule和test两个文件夹

rm

删除一个目录中的一个或多个文件或目录，如果没有使用- r选项，则rm不会删除目录。

如果使用 rm 来删除文件，通常仍可以将该文件恢复原状

可选参数

  • -f (—force) 强制删除,忽略不存在的文件,从不给出提示
  • -r, -R (—recursive) 递归(子目录)删除
  • -i (—interactive) 交互式删除

参数组合使用 (rm -rf)

rmdir

该命令的功能是删除空目录

rm - r dir 命令可代替rmdir, 但是有危险性

参数

  • -p 递归删除,当子目录删除后其父目录为空时,也一同被删除

mv

mv命令是move的缩写，可以用来移动文件或者将文件改名

根据mv命令中第二个参数类型的不同(是目标文件还是目标目录), mv命令将文件重命名或将其移至一个新的目录中

当第二个参数类型是文件时，mv命令完成文件重命名

Ex: mv test1.txt test1.log

当第二个参数是已存在的目录名称时，源文件或目录参数可以有多个，mv命令将各参数指定的源文件均移至目标目录中

Ex:

  • mv test1.log TestFiles 将test1.log移动到testFiles文件夹
  • mv test1.log test2.log TestFiles 将test1.log和test2.log移动到testFiles文件夹
  • mv TestFiles TestFiles1 将TestFiles目录移动到TestFiles1目录,如果TestFiles1不存在,将目录TestFiles改名为TestFile1
  • mv * ../ 移动当前文件夹下的所有文件到上一级目录

常用参数

  • -b 若需覆盖文件，则覆盖前先行备份

mv log1.txt -b log2.txt 当目录中已经存在log2.txt 则在覆盖前,进行简单备份

cp

cp命令用来复制文件或者目录

常用参数

  • -r, -R 递归复制目录中的所有项目
  • -i 覆盖前进行询问
  • -n 不要覆盖已存在的文件

touch

用来修改文件时间戳，或者新建一个不存在的文件

Ex: touch log1.log log2.log 创建两个不存在的文件

cat

cat命令的用途是连接文件或标准输入并打印。这个命令常用来显示文件内容

Ex: cat test.log 在命令行输出文件内容

pwd

查看”当前工作目录“的完整路径

echo

向文件写入内容

Ex:

覆盖型写法: echo "aaa" > log.txt

添加型写法: echo "bbb" >> log.txt

More…

  • vim
    • 在mac上, :w保存 :wq保存关闭 :q关闭 :q!强制关闭
    • 在Windows上需要先按下Esc键结束插入才行
